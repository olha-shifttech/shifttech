# ShiftTech Content Templates



## 1. Назва документа



05_Content_Templates.md



Статус: Draft



Версія: 1.0



Власник: ShiftTech



Призначення: зафіксувати шаблони сторінок для сайту, бази знань, AI-пошуку та майбутньої генерації контенту.



---



## 2. Головна мета



Створити єдині шаблони сторінок, щоб кожен новий матеріал ShiftTech мав однакову логіку, структуру і якість.



Цей документ потрібен для:



- сайту;

- бази знань;

- AI-агентів;

- Codex;

- SEO;

- Google Search;

- ChatGPT / Claude / Gemini / Perplexity.



---



## 3. Головний принцип



Кожна сторінка повинна бути зрозумілою трьом аудиторіям:



1. Клієнту — просто, без зайвої технічної складності.

2. Майстру — точно, структуровано, з технічними деталями.

3. AI — чітко, з явними сутностями і зв'язками.



---



## 4. Обов'язкові типи шаблонів



Перший пакет шаблонів:



1. Vehicle Page — сторінка автомобіля

2. Transmission Page — сторінка коробки передач

3. Symptom Page — сторінка симптому

4. Error Code Page — сторінка помилки

6. FAQ Page — сторінка питання-відповіді

---



# 13. Заборонено



Заборонено:



- писати сторінки без шаблону;

- створювати сторінки без внутрішніх зв'язків;

- дублювати один і той самий факт;

- змішувати кілька тем в одну сторінку;

- писати тільки для SEO без користі;

- використовувати неперевірені факти як істину;

- робити складний текст для клієнта без простого пояснення.



---



# 14. Правило якості



Сторінка якісна, якщо:



1. Клієнт розуміє, що з його авто може бути.

2. Майстер бачить технічну логіку.

3. AI розуміє зв'язки між сутностями.

4. Є шлях до наступної дії.

5. Немає дублювання інформації.
